# Wrex.app

Wrex.app is a clean, modern educational AI-text analysis platform for students.

## Phase 1

- Free detector
- Locked Pro writing preview
- Waitlist
- Educational disclaimer

## Stack

- React + Tailwind CSS frontend
- FastAPI backend

## Frontend

```bash
npm install
npm run dev
```

## Backend

```bash
cd backend
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

## Product Tone

- Educational
- Fair
- Probabilistic
- Not punitive
